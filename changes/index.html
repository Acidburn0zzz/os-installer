Evolve OS Alpha 3 August 3rd, 2014

Shiny kernel

This version of Evolve OS includes a very recent kernel version, 3.15.6, ensuring better hardware support.

New X.Org Stack

We've included X.Org 1.16.0 and MesaLib 10.2.4 in this release, again improving hardware support and improving performance across the board. All drivers have been updated to their latest versions, and LLVM3.4.2 is available by default to assist in non-GPU accelerated cases

Updated to the GNOME 3.12.2 stack

This release now includes the GNOME 3.12 stack, offering a far more polished experience. In September you'll automatically be updated to GNOME 3.14

The last alpha..

This release represents the last alpha we're publishing. We're moving into daily(-ish) builds until beta1, vastly accelerating the development rate of Evolve OS
