Evolve OS Alpha 2 July 13th, 2014

A new version of Budgie

This release includes Budgie v4, which now features a rundialog. You'll receive updates for Budgie Desktop first, as both Evolve OS and the Budgie desktop are developed in tandem.

A few good taters..

As well as addressing some basic bugs, we've included Gimp 2.8.10 for your pleasure, as well as GNOME Maps. Because knowing where you are is awesome.

Installer fixes

The installer now requires a lower-case username, as well as a minimum password length of 6 characters. In a future version of the installer guidelines will be clearly visible

Moar integration

The desktop experience is somewhat more integrated now, enabling the use of traditional keybindings for the panel menu (Super_L) and the run dialog (ALT+F2).
